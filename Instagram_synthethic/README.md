# Instagram Synthetic Cyberbullying Dataset

## Dataset Type
Author-Created Synthetic Dataset

## Platform Simulated
Instagram

## Data Type
Multimodal (Text + Image)

## Dataset Description
This dataset was generated to simulate modern multimodal cyberbullying behavior on Instagram-like platforms.

It contains:
- Image-text paired posts
- Synthetic captions and comments
- Binary labels (Bullying = 1, Non-bullying = 0)

## Dataset Size
5,236 samples

## Bullying Ratio
23.8%

## Purpose
Created to:
- Expand multimodal diversity
- Simulate cross-platform bullying behavior
- Test model generalization

## Data Generation Method
- Synthetic augmentation techniques
- Controlled class balancing
- Manual validation of sample realism

## Public Availability
This dataset is not sourced from a public platform.
It was created exclusively for research experimentation.

## Ethical Consideration
No real Instagram user data was scraped or redistributed.
All samples were synthetically generated for academic purposes.

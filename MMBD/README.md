Twitter

## Data Type
Multimodal (Text + Image)

## Dataset Description
The MMBD dataset contains paired Twitter posts consisting of:
- Tweet text
- Associated image
- Binary label (Bullying = 1, Non-bullying = 0)

The dataset was designed to evaluate multimodal cyberbullying detection models.

## Subset Used in This Study
- Total Samples Used: 8,472
- Bullying Ratio: 25.4%
- Used for baseline reproduction experiments.

## Access Information

- Request tweet ID list from authors
- Hydrate tweets using Twitter API

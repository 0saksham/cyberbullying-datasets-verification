## Dataset Name

## Source Type
Published Open-Access Dataset (External Source)

## Original Publication
Van Hee, C., Lefever, E., Verhoeven, B., Mennes, B., Desmet, B., Hoste, P., & De Clercq, V. (2018).

## Platform
Twitter

## Data Type
Text-only

## Dataset Description
This dataset consists of annotated Twitter posts labeled for cyberbullying content.
It includes:
- Tweet text
- Cyberbullying category labels
- Manual human annotations

## Approximate Size
~10,000 tweets (as referenced in the study)

## Usage in This Study
Used for:
- Supplementary text-based experiments
- Model comparison and validation

## Access Information
The dataset description and annotation methodology are available in the open-access PLOS ONE publication.
Actual tweet content may require access via tweet IDs and Twitter API hydration.

## Redistribution Policy
Raw tweets are not redistributed in this repository in compliance with Twitter policies.

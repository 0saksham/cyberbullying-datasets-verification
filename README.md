# Datasets Used in MultiBully-XAI Research

This repository documents all datasets used in the MultiBully-XAI framework for verification purposes.

---

## 1. Twitter Cyberbullying Dataset (Van Hee et al., 2018)

Source:
https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0203794

Details:
- Platform: Twitter
- Type: Text-only
- Approx. 10,000 tweets
- Used for baseline text experiments

Note:
Dataset may be accessed through supplementary material or by contacting authors.

## 3. Instagram Synthetic Dataset

Details:
- 5,236 multimodal samples
- Created for research purposes
- 23.8% bullying ratio
- Generated using synthetic augmentation

Not publicly sourced.

---

## 4. XAI Evaluation Dataset

Details:
- 200 manually labelled samples
- Used for SHAP, LIME, Attention evaluation

Created internally.

---

## 5. Real-World Validation Dataset

Details:
- 150 samples
- Platforms: Twitter, Instagram, Facebook, TikTok
- Annotated by 3 moderators
- Majority voting used

Created for validation testing.

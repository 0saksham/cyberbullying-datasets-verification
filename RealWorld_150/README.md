# Real-World Social Media Validation Dataset

## Dataset Type
Author-Collected Validation Dataset

## Dataset Size
150 samples

## Platforms Included
- Twitter
- Instagram
- Facebook
- TikTok

## Data Type
Multimodal (Text + Image)

## Purpose
Used to validate model generalization on real-world uncontrolled data.

## Annotation Process
- Each sample reviewed by 3 independent human moderators
- Majority voting applied for final label
- Conflict resolution performed via discussion

## Performance Evaluation Categories
- Explicit cyberbullying
- Implicit cyberbullying
- Multimodal bullying
- Non-bullying

## Ethical Considerations
- No private data disclosed
- No user-identifiable information stored
- Data used strictly for research validation

## Public Availability
This dataset is not publicly redistributed.
It was manually curated for experimental validation.
